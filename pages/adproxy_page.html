🚀 MTProto Proxy Setup

Генератор команд для установки прокси с рекламой

Например: mtproto-proxy, proxy1, tg-proxy
Рекомендуется 443 (HTTPS) или 8080
32 символа от @MTProxybot (шаг 4)

📋 Пошаговые команды

1
Подготовка сервера
Установите Docker, если ещё не установлен:
curl -sSL https://get.docker.com/ | CHANNEL=stable bash
2
Первый запуск (генерация секрета)
Запустите контейнер для генерации секретного ключа:
docker run -d \ -p 443:443 \ --name=mtproto-proxy \ --restart=always \ -v proxy-config:/data \ telegrammessenger/proxy:latest
⚠️ После запуска выполните: docker logs mtproto-proxy
Скопируйте секрет (32 символа) и зарегистрируйтесь в @MTProxybot
3
Остановка старого контейнера
Перед перезапуском с TAG остановите старый контейнер:
docker stop mtproto-proxy && docker rm mtproto-proxy
4
Финальный запуск с рекламой
Запустите прокси с рекламным тегом:
docker run -d \ -p 443:443 \ --name=mtproto-proxy \ --restart=always \ -v proxy-config:/data \ -e TAG=ваш_тег_от_бота \ telegrammessenger/proxy:latest
✅ Тег подставлен! Скопируйте команду выше.
5
Проверка и получение ссылки
Посмотрите логи для получения ссылки на подключение:
docker logs mtproto-proxy
В логах найдите ссылки вида:
tg://proxy?server=IP&port=443&secret=...
⚡
Дополнительные команды
Перезапуск контейнера:
docker restart mtproto-proxy
Просмотр статуса:
docker ps -a | grep mtproto-proxy
Удаление контейнера и тома (полная очистка):
docker stop mtproto-proxy && docker rm mtproto-proxy && docker volume rm proxy-config
