Например: tg-proxy, proxy1, mtproto
Генерация секрета
Нажмите кнопку ниже чтобы сгенерировать секретный ключ, или введите свой:
Ваш секретный ключ (Secret)
32 шестнадцатеричных символа (0-9, a-f)
Регистрация в @MTProxybot
Данные для регистрации:
Отправьте боту /newproxy, затем введите:

Хост:порт: -
Секрет: -
После регистрации бот выдаст рекламный тег (опционально):
Оставьте пустым чтобы запустить без рекламы

📋 Пошаговая установка

1
Установка Docker
curl -sSL https://get.docker.com/ | CHANNEL=stable bash
2
Создание директории
mkdir -p ~/tg-proxy/tg-proxy && cd ~/tg-proxy/tg-proxy
3
Запуск контейнера
docker run -d \ --name tg-proxy \ -p 443:443 \ -v $(pwd)/data:/data \ -e SECRET=ваш_секрет \ -e TAG=ваш_тег \ --restart=always \ telegrammessenger/proxy:latest
⚠️ Введите секрет для формирования команды запуска
4
Проверка логов
docker logs -f tg-proxy
5
Готовая ссылка для подключения
🔗 Ссылка для Telegram
⚠️ Заполните все поля (IP/домен, порт, секрет) чтобы сформировать ссылку

🛠️ Дополнительные команды

⚡
Управление контейнером
Перезапуск:
docker restart tg-proxy
Остановка:
docker stop tg-proxy
Запуск остановленного:
docker start tg-proxy
Просмотр статуса:
docker ps -a | grep tg-proxy
Удаление контейнера (данные сохранятся):
docker stop tg-proxy && docker rm tg-proxy
Полная очистка (с удалением данных):
docker stop tg-proxy && docker rm tg-proxy && rm -rf ~/tg-proxy/tg-proxy
