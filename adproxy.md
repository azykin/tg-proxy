## Самая простая установка MTProto прокси с рекламой 
*(официальный образ Telegram)*

### Шаг 1. Подготовка сервера
Понадобится VPS (виртуальный сервер) с публичным IP-адресом и установленным Docker.
Если Docker еще не установлен, выполните на сервере команду:

`curl -sSL https://get.docker.com/ | CHANNEL=stable bash`

### Шаг 2. Запуск прокси (без рекламы)
Сначала запустите контейнер, чтобы он сгенерировал секретный ключ. Этот ключ понадобится для регистрации.
Выполните команду, заменив `YOUR_PORT` на желаемый порт (например, 443):

```
docker run -d  \
  -p YOUR_PORT:443  \
  --name=mtproto-proxy  \
  --restart=always  \
  -v proxy-config:/data  \
  telegrammessenger/proxy:latest
```

- `-p YOUR_PORT:443` — пробрасывает порт вашего сервера на внутренний порт контейнера
- `-v proxy-config:/data` — создает том для хранения конфигурации (ключа)

### Шаг 3. Получение секретного ключа
После запуска контейнера посмотрите его логи, чтобы узнать сгенерированный секрет:

```
docker logs mtproto-proxy
```

В выводе вы увидите строку, похожую на: `Secret 1: 00baadf00d15abad1deaa515baadcafe`.
**Скопируйте этот секрет** (32 символа) — он понадобится на следующем шаге и для подключения пользователей.

### Шаг 4. Регистрация прокси и получение рекламного тега (TAG)
Чтобы добавить рекламу (спонсорский канал), нужно зарегистрировать свой прокси-сервер в Telegram.

1. Откройте Telegram и найдите официального бота: [@MTProxybot](https://t.me/MTProxybot).
2. Запустите его и отправьте команду `/newproxy`.
3. Бот попросит ввести данные вашего сервера:
   - **IP-адрес** вашего сервера.
   - **Порт**, который вы указали при запуске (`YOUR_PORT`).
   - **Секрет**, который вы скопировали на предыдущем шаге.
4. После успешной регистрации бот выдаст вам уникальный тег (TAG) — строку из 32 символов (например, `3f40462915a3e6026a4d790127b95ded`). Этот тег и отвечает за показ рекламы.

### Шаг 5. Финальный запуск прокси с рекламой
Теперь нужно остановить и удалить старый контейнер и запустить новый, передав ему полученный тег.

```
docker stop mtproto-proxy
docker rm mtproto-proxy
```
Запустите новый контейнер, добавив переменную окружения TAG:

```
docker run -d \
  -p YOUR_PORT:443 \
  --name=mtproto-proxy \
  --restart=always \
  -v proxy-config:/data \
  -e TAG=ваш_тег_от_бота \
  telegrammessenger/proxy:latest
```

**Важно:** Секрет хранится в томе proxy-config и не будет потерян, а тег нужно передавать при каждом создании контейнера .

### Шаг 6. Подключение к прокси
Чтобы подключиться к вашему новому прокси с рекламой, снова посмотрите логи контейнера:

```
docker logs mtproto-proxy
```
Найдите в логах ссылки для подключения. Они будут выглядеть так:

- `tg://proxy?server=YOUR_IP&port=YOUR_PORT&secret=ваш_секрет`
- `https://t.me/proxy?server=YOUR_IP&port=YOUR_PORT&secret=ваш_секрет`

Просто скопируйте одну из них и отправьте себе в Telegram (например, в "Избранное"). При переходе по ссылке Telegram сам предложит настроить прокси.

### Проверка
После подключения к прокси у пользователей в списке чатов может появиться канал, который вы указали при регистрации (или который выдал бот), и они не смогут его отписать, пока используют ваш прокси.
